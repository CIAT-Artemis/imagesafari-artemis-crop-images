# Changelog

All notable changes to the Artemis and Image Safari public releases are
documented in this file.

## [Unreleased]

### Documentation

- Rename annotation track `automated` to `standard` for clarity.
- Add metadata dictionary, annotation guidelines, validation report template,
  and split documentation.
- Clarify FAIRGrounds as the primary release platform with AWS Open Data mirror
  and GitHub-hosted schemas and tutorials.

## [1.0.0] - TBD

### Added

- Initial public release of Artemis (1,387,665 images across four crops).
- Initial public release of Image Safari (6,081,555 images across 18 crops).
- Annotated subset of 42,981 images with bounding box, mask, point, scribble,
  and polygon labels.
- Benchmark annotation track for banana, common bean, potato, and sorghum.
- Predefined evaluation splits stratified by location, acquisition method,
  date, and growth stage.
